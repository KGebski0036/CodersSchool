# Calculate

simple calculator with tests.

## How to build

*When you are in `calculate` directory.*

```bash
mkdir build
cd build
cmake ..
make
./calculate
GTEST_COLOR=1 ctest -V
```
or replace last command with
```
ctest
```
for simpler output
